
# कैलकुलेटर

सरल कैलकुलेटर: तेज़ और आसान हिसाब-किताब। दैनिक उपयोग के लिए एक बेसिक और साफ़ कैलकुलेटर। जोड़, घटाव, गुणा और भाग करें बिना किसी परेशानी के। यह हल्का है और बहुत तेज़ काम करता है।

## विशेषताएं

- लाइट मोड टॉगल
- सरल गणना
- सरल और साफ़ UI
